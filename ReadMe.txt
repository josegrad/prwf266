### TicTacToe with HTML5 Offline Storage ###
 
===========================================================================
DESCRIPTION:
 
This sample uses HTML5 localStorage to store the state of a tic-tac-toe game, and to restore the state of the game the next time this page is visited.  You can close the window or even quit Safari, and the state of game will be saved.  You can inspect the stored data for the game in Safari's Web Inspector.

===========================================================================
BUILD REQUIREMENTS:
 
Safari 4.0 and later, Safari on iOS OS 3.0 and later
 
===========================================================================
RUNTIME REQUIREMENTS:
 
Safari 4.0 and later, Safari on iOS OS 3.0 and later
 
===========================================================================
PACKAGING LIST:
 
tic-tac-toe.html - HTML file for the sample web page.
main.js - JavaScript that creates the game board, handles game logic, and stores and restores the state of the game
main.css - Basic styles for the page - nothing specific to HTML5 offline storage
